Project 1

Pi-hole Netwok ad blocker

Installed a DNS sinkhole for my home network using a Raspberry Pi 3B+ and software called “Pi-hole”. The software blocks ads and malware from my entire home network by filtering a blocklist over both IPv4 and IPv6.

Project 2

OctoPrint 3D sofware

Connected OctoPrint for my Ender 3S1, a network-based software tool that allows me to remotely control my 3D printer using a Raspberry Pi 3B+. I also attached a camera, allowing me to create timelapses of my prints and remotely watch the printer while away from home.
